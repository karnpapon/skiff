promo website for launching first album of The ███████

The Black Codes is an electronic bangkok-based musical act, consists of Karnpapon Boonput. the website is intended to be a part of promoting the first album “Boring Original Nonsense” ( 2018, self-released). the website is simply SPA (Single-page application) meets minimal design. distorted reflection ( using webgl) via webcam on the first page simulate the physical release where the holographic sticker was pasted on the transparent plastic cover.
