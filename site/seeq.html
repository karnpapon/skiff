a custom sequencer, draw an inspiration from String‚Äôs definition.

String is traditionally a sequence of letters. Seeq is yet-another-live-coding-environment-alike. It's heavily inspired by Orca. Initially, the idea coming from being tired of keep remembering new functions from TidalCycles.

Don't get me wrong. TidalCycles is hands-down obviously a great language out there. But for non-programming person like me. I cannot utilize the true power of TidalCycles without knowing Haskell language.

Thus, for the sake of understanding programming in general. I decided to roll my own. JavaScript is a language of choice since it's personally more friendly for newbie. Creating programming language is considered advanced topic. So, I decided to looking for practical way like sequencer instead.

Usage

to play with it, just put any words, sentences, just like searching from Google or any search engine. Seeq will manage to find the descriptions for that keywords and use it as a step sequencer (using API from WikiPedia).

a ‚Äútrigger‚Äù is simply assigned by typing any words/letters in provided input, Regex is also available. currently support only sending MIDI out.

in order to move selection, the console has to be toggled off ( cmd + i).
