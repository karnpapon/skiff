The Black Codes is an electronic bangkok-based musical act, consists of [Karnpapon Boonput](/site/about.html). the website is intended to be a part of promoting the first album [“Boring Original Nonsense”](/site/the-blackcodes-artworks.html) ( 2018, self-released). the website is simply SPA (Single-page application) meets minimal design. distorted reflection ( using [webgl](https://veda.gl/vedajs/)) via webcam on the first page simulate the physical release where the holographic sticker was pasted on the transparent plastic cover.
