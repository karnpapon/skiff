calendrical sequencer based on Github's contribution history. where velocity of the note is determined by commit's frequency. total 53 weeks, the only way to assign trigger is just commit!.

features

  • MIDI protocol (out only)
  • Internal Synth voice (13 instruments)
  • adjustable
  • steps length.
  • scale / root note.
  • pitch.
  • midi-channel.
  • volumn ( only works for internal synth).
  • octave.
  • BPM
  • track mute.
  • step mute.

    • see this on github : https://github.com/karnpapon/patithin
